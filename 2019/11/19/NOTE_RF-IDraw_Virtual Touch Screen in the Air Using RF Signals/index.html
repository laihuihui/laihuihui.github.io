算法流程：

  1. 根据计算找出几个候选点作为标签最初位置（利用天线对5678，找票数 vote 最高的几个点作为候选点）
  2. 对每个候选点，根据确定该候选点的光栅的旋转进行连续追踪
  3. 估计下一个点的位置时，注意 ensuring continuity when angles wrap around [0, 2π].
  4. 计算整个轨迹各个点的 vote 总和，找出总和最大的候选点（利用所有天线对，即 <1,2><1,3><1,4><2,3><2,4><3,4><5,6><7,8><5,7><5,8><6,7><6,8> ）
