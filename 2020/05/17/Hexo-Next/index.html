【博客搭建】Hexo+NexT

发表于 2020-05-17 分类于 BLOG Valine：
本文字数： 938 阅读时长 ≈ 1 分钟

Hexo是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

NexT是用于 Hexo 的一款简洁风主题。

码云是 OSCHINA.NET 推出的代码托管平台，支持Git 和SVN，提供免费的私有仓库托管。

Hexo

  • 参照官网教程进行安装
  • 参照命令解释文档进行站点初始化、部署、服务器启动
  • 浏览器访问 http://localhost:4000/，出现新建的网站

NexT

根据 NexT 文档 进行安装和配置 (可以参考旧版本文档)

Gitee 部署

  • 开启 Gitee Pages 服务

  • 配置文档 Hexo\_config.yml，url设置为启动仓库的Gitee Page服务后分配的网站静态域名，同时设置root

  • 配置文档 Hexo\_config.yml，repository为仓库地址 (clone 时候的地址)

  • 部署

    • 方式1：用 hexo d 命令
    • 方式2：用 git push 命令

  • 【ATTENTION】免费Gitee用户需要手动更新 Gitee Page，否则网站内容不会发生变化
